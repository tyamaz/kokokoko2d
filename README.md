kokokoko2d.nvim
================================================================================
Something to visualize cursor moving.
kokokoko.nvim implements by sign column.
The plugin by overlay text.

DIY!

Thanks.

Special Feature
--------------------------------------------------------------------------------

Install
--------------------------------------------------------------------------------
### for lazy.nvim

```lua
{
  "tyamaz/kokokoko2d.nvim",
  config = function()
    require("kokokoko2d").setup({
      speed = 300
    })
  end
}
```
